# 前端面试手写题库

> 系统化的前端核心知识点手写实践仓库，涵盖JavaScript API、工具函数、React组件、算法与设计模式等。

## 📂 项目结构

```
Handwritten-practice/
├── 01-手写API/              # JavaScript 核心 API 实现
│   ├── Object相关/          # Object.create, instanceof, new 等
│   ├── Promise相关/         # Promise.then, Promise.all, Promise.race
│   ├── Function相关/        # call, apply, bind
│   └── Array相关/           # filter, reduce, map 等
│
├── 02-手写工具函数/          # 通用工具函数实现
│   ├── 性能优化/            # 防抖、节流、并发控制、请求缓存、时间分片
│   ├── 数据处理/            # 深拷贝、柯里化、扁平化、compose
│   ├── 业务工具/            # 千分位转换、版本号比较、数字转中文、LazyMan
│   └── __tests__/          # 单元测试文件
│
├── 03-手写组件/              # React 组件实现
│   ├── 基础组件/            # 倒计时、弹窗、级联选择、数字输入框、Select 等
│   └── 带环境的组件/        # 完整的 React 项目环境
│
├── 04-算法/                 # 算法与数据结构
│   ├── 动态规划/            # 最小编辑距离、打家劫舍等
│   ├── 双指针or滑动窗口/    # 三数之和、最小覆盖子串、最长无重复字串
│   ├── 树的遍历/
│   │   ├── BFS/            # 广度优先搜索
│   │   ├── DFS/            # 深度优先搜索
│   │   └── 回溯法/          # 全排列、组合总和等
│   ├── 贪心/                # 合并区间等
│   ├── 链表/                # 环形链表等
│   └── 队列和栈/            # 优先级队列、括号匹配等
│
├── 05-设计模式/             # 常见设计模式实现
│   ├── 发布订阅模式
│   ├── 观察者模式
│   ├── 单例模式
│   ├── 工厂模式
│   ├── 策略模式
│   ├── 装饰器模式
│   └── 中间件模式
│
├── 06-CSS/                 # CSS 布局与技巧
│   └── flex布局
│
└── 07-代码执行题/           # 代码执行与测试题
    ├── this 指向
    └── React 相关
```

## 🎯 使用说明

### 快速开始

```bash
# 克隆仓库
git clone <repo-url>

# 进入项目目录
cd Handwritten-practice

# 查看具体实现
cd 01-手写API/Promise相关
```

### 运行组件示例

```bash
# 进入组件环境
cd 03-手写组件/带环境的组件

# 安装依赖
npm install

# 启动开发服务器
npm start
```

## 🔖 文件命名规范

- 所有文件使用小写 `.js/.jsx/.tsx` 扩展名
- 函数类文件以功能命名（如：`防抖函数.js`）
- 测试文件统一放在 `__tests__/` 目录
- 算法题以题目名称命名（如：`三数之和.js`）

## 💡 学习建议

1. **循序渐进**：按目录顺序学习（API → 工具函数 → 组件 → 算法 → 设计模式）
2. **动手实践**：每个文件都包含完整实现，建议先自己尝试再查看
3. **理解原理**：不仅要会写，更要理解背后的原理
4. **查看模板**：各算法目录下都有思路模板文档（如 `回溯法模版.md`）
