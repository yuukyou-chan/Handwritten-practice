## 动态规划思路和模板
动态规划一般解决的是最值问题：如最小编辑距离、最长递增子序列、最长回文子序列等。
### 思路
核心是写出状态转移方程：即当前状态和之前状态的关系。
### 模板
```
// 1. 初始化 base case
dp[0][0][...] = base case
// 2. 穷举状态
for 状态1 in 状态1的所有取值：
    for 状态2 in 状态2的所有取值：
        for ...
            // 3. 进行状态转移
            dp[状态1][状态2][...] = 求最值(选择1，选择2...)
```
